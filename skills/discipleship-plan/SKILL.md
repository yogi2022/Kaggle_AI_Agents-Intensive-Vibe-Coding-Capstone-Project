# Discipleship Plan Skill

Turns biblical insight into a doable plan. Use after the issue is diagnosed.

## Output shape
A plan with, for each day:
- **Scripture** — one grounded passage (from a Scripture tool).
- **Reflection** — one short question.
- **Prayer** — one or two sentences.
- **Action step** — one concrete, small, achievable action.
- **Accountability** — one yes/no check for the next day.

## Rules
1. Keep daily steps small enough to actually complete (5-15 minutes).
2. Sequence the days so they build (awareness -> truth -> practice -> habit).
3. Every Scripture must come from a tool result (see scripture-citation skill).
4. End with one weekly review question.
5. For a 30-day plan, group into four weekly themes and reuse this daily shape.
